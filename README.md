Linux Keyboard Debouncer

**Requirement**
- libevdev
- interception-tools
- cmake

**How To Build & Install**
```bash
git clone https://github.com/Rliop913/Linux-Key-Debouncer.git
cd Linux-Key-Debouncer
bash ./debounceSetter.sh
```

**How To Activate**
```bash
sudo bash ./runDebounce.sh /dev/input/eventX
```

Done!
